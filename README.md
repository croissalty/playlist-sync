# playlist-sync

playlist sync is a personal automation tool that keeps a spotify playlist and a youtube playlist synchronized.

the app compares playlists between spotify and youtube and identifies songs that exist on one platform but are missing from the other.

when a song exists in the spotify playlist but not in the youtube playlist, playlist sync searches youtube for a matching song and adds it to the youtube playlist.

when a song exists in the youtube playlist but not in the spotify playlist, playlist sync searches spotify for a matching song and adds it to the spotify playlist.

the goal is to keep both playlists as similar as possible so they contain the same music across both platforms.
